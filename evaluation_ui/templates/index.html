📋 Evaluation Instructions

Thank you for contributing to this research. Please evaluate each AI-generated solution based on your professional judgment.

For each task, you will rate:
  • Correctness (1-5): Is the solution logically and functionally correct?
  • Readability (1-5): Is the code clear, well-structured, and easy to understand?
  • Intention to Use: Would you adopt this code in a project?
  • Comments (Optional): Any specific feedback on style, bugs, or improvements.
Evaluation Principles:
  • Consider the solution as if you encountered it in a real code review.
  • Base your rating on the provided Problem Prompt (the task specification).
  • You can use the "Previous" button to revisit past problems before submitting your final ratings.
  • Your responses will be saved automatically after clicking "Submit & Next Problem".
